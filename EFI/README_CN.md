## 工作的部分

- macOS 10.15.6（支持11.0 Big Sur）
- CPU（睿频4.0Ghz）
- 核芯显卡
- 有线网卡
- 音频（Layout=15）
- USB（定制USBPorts.kext）
- 触摸板
- 摄像头
- 蓝牙（带有蓝牙开关）
- Wi-Fi（使用[itlwm](https://github.com/OpenIntelWireless/itlwm))

## 不工作的部分

- 独立显卡（已使用SSDT屏蔽）
- HDMI（直接连接到独立显卡）
- 内置读卡器
